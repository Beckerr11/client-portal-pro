# Client Portal Pro

Portal de cliente para gestao de projeto com transparencia total.

## Problema
Clientes ficam sem visibilidade de progresso em canais fragmentados.

## Publico
Freelancers, consultorias e agencias de software.

## MVP
- Timeline de marcos com status de entrega.
- Canal de mensagens e anexos por projeto.
- Fluxo de aprovacao e revisao de entrega.
- Area de documentos e contratos.

## Stack
- React + TypeScript + Vite
- Node + Express + TypeScript
- CI com lint/test/build/audit

## Setup
1. npm install --include=dev
2. npm run bootstrap
3. npm run dev

## Monetizacao
Plano mensal por clientes ativos.
